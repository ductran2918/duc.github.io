🧪 Testing SEA Founders Map Embed

This page tests if your embed script works correctly before using it in your CMS.

✅ What should happen:

  • You should see your interactive map embedded above
  • It should be responsive and properly sized
  • All interactions (dropdown, clicking states) should work
  • It should look the same as your original page

🐛 If it doesn't work:

  • Open browser developer tools (F12)
  • Check the Console tab for error messages
  • Make sure the BASE_URL in embed.js is correct
